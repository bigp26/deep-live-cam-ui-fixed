# Deep Live Cam UI

This repository contains the UI component for the Deep Live Cam project.

## Features
- CustomTkinter based interface
- Image and Video face swapping support
- Live webcam support
- Face mapping and enhancement options

## Setup
Ensure you have the necessary dependencies installed:
```bash
pip install customtkinter opencv-python pillow numpy
```

Note: This module depends on a `modules` package containing `globals`, `metadata`, `face_analyser`, etc.
